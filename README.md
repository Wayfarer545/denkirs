# Denkirs for Home Assistant

Local control of **Denkirs** smart track lighting in Home Assistant — no cloud at
runtime, no vendor account required after setup.

Denkirs SMART fixtures are BLE-mesh nodes behind a Wi-Fi gateway. This integration
talks to the gateway over the local Tuya LAN protocol and addresses each fixture by
its mesh id, so brightness and colour-temperature control stay entirely on your
network.
